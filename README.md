# 📝 Todolist App

A simple full-stack task manager app with authentication.

## 🚀 Tech Stack
- **Frontend:** React + Vite + TypeScript
- **Backend:** Node.js + Express + Prisma
- **Database:** PostgreSQL
- **Auth:** JWT stored in secure HTTP-only cookies
